# Image Resizer

## Setup

- Clone repo `git clone https://github.com/wanderer-moe/img-resizer`
- Modify `wrangler.toml` to your needs
- Deploy w/ `wrangler deploy`

## Run locally

- `wrangler dev`

## Deploy

- `wrangler deploy`
